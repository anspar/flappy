Tutorials -> [Youtube](https://www.youtube.com/embed/6gErdx-llSg?list=PLynQF5IaLl6HypAzcdlxSqBdPshRNXtS6)

## To run the game use

    arag -s

## To package the game use

    arag --pkg

Use the resulting `build.html` file to mint the dapp on [anspar.io](https://anspar.io)
